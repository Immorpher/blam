# Basic Line Action Macro Compiler (BLAM)
BLAM there it is! This is a Doom 64 macro compiler originally developed by [Kaiser](https://sourceforge.net/p/doom64ex/code/HEAD/tree/trunk/src/blam/).


## Usage
Usage: <blam.exe> <scriptfile> <outfile> <option>


Options:

-d	Decompile script lump

-i	Set directory to search for files to include


## Compiling
This is compiled via Microsoft Visual Studio.
